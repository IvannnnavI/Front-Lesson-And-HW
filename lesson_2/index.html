CSS можно подключить тремя разными способами:

  • Первый - внутренний. Мы указываем код в теге style
  • Второй способ - инлайновый. Когда мы прописываем стили внутри открывающего тега
  • Третий (приоритетный) - создать файл со стилями CSS

инлайновый CSS имеют найвысший приоритет, но старайтесь не использовать их
